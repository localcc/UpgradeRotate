# UpgradeRotate

A [Mycopunk](https://store.steampowered.com/app/3247750/Mycopunk/) mod that allows upgrades to be rotated in the hex grid.

## Installation

* [Install MelonLoader](https://melonwiki.xyz/#/README?id=automated-installation)
* Launch the game at least once after installing MelonLoader
* Download the [lastest release](https://github.com/localcc/UpgradeRotate/releases/latest) of the mod
* Drag and drop the `.dll` file into `GAMEFOLDER/Mods/`
* Enjoy!

## Usage

Once the mod is installed, you can open the player/weapon upgrades menu and rotate your upgrades with

* Keyboard - `Q`/`E` for left/right rotation, `R` to reset the rotation.
* Gamepad - `Left Bumper`/`Right Bumper` for left/right rotation, to reset the rotation press `Square` on PlayStation controllers and `X` on Xbox controllers.
